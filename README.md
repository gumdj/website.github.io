## 一、项目主题

​	本项目采用的Vue.js框架设计的动态前端页面，作用于商店管理系统的前端页面。注：（github没有数据显示，因为数据显示需要后端提供接口）



## 二、开发过程及介绍

### 2.1 选择的框架和工具

​	我采用了主流的前后端分离式开发，以Vue.js作为整个项目的框架并用mock.js模拟后端，axios.js处理前端发出的请求，嵌入element-ui作为界面框架，idea作为编写的ide。
